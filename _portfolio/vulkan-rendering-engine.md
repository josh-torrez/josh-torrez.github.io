## Overview

Currently developing a modern graphics engine using the Vulkan API, focusing on cross-platform compatibility and low-level graphics programming. The engine targets macOS through MoltenVK while maintaining best practices for graphics API usage.

## Key Features

- **Vulkan API Integration**: Low-level graphics programming with validation layers
- **Cross-Platform Support**: MoltenVK implementation for macOS compatibility
- **Device Management**: Physical device selection and queue family management
- **Modern Build System**: CMake configuration with Apple Silicon optimization

## Technologies

- C++
- Vulkan API
- GLFW
- CMake
- MoltenVK

## Technical Highlights

The project emphasizes proper Vulkan usage patterns with comprehensive validation layer integration for debugging. The CMake build system is configured specifically for Apple Silicon, ensuring optimal performance on modern Mac hardware.
